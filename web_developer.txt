Web developer adalah seseorang yang bertugas membuat dan mengembangkan website atau aplikasi web agar bisa berjalan dengan baik saat diakses melalui internet.

Jenis-jenis Web Developer:
Frontend Developer
Fokus pada bagian yang dilihat dan digunakan langsung oleh pengguna (UI/UX).
Contoh: tampilan tombol, navigasi, warna, layout, dsb.
Tools umum: HTML, CSS, JavaScript, React, Vue, dll.

Backend Developer
Mengurus bagian belakang layar: logika server, database, autentikasi, API, dll.
Tools umum: PHP (Laravel), Node.js, Python (Django), MySQL, MongoDB, dsb.

Fullstack Developer
Gabungan frontend dan backend â€” bisa mengerjakan dua-duanya.


Tugas Utama Web Developer:
Membangun halaman website atau aplikasi web.

Menyambungkan website dengan database.

Memastikan tampilan responsif di berbagai device.

Menyelesaikan error/bug dan menjaga performa web.

Berkolaborasi dengan tim (designer, tester, project manager, dll).
